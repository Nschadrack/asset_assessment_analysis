Before filling out the form for recording a new Physical node, here is the explaination of some fields for supporting you to fill out with required and correct data about Physical node

SERIAL NUMBER: These are series of characters used to identify a Physical node

Tag number:This is a series of characters which identifies each asset of an organization

Entry date in the bank: The date when the bank has received the Physical node from supplier

Warranty time(in months): The time given by the supplier to the bank for Physical node to be returned back if it doesn't meet the specifications

Life time (in months): The time the Physical node is supposed to be used and after that time the computer should not be reliable(provided by the manufacturer and shuould be followed)

Store: If the Physical node is in stock which stock where it is stored

Purchase price(RWF): Amount that spent while buying Physical node

Sell price: If the Physical node was sold, at which price it was sold.

Working status: Behaviour of the Physical node while being used

Category: The type of Physical node whether TV or monitor

Physical node size: The size of the Physical node in inches
