Before filling out the form for recording a new scanner, here is the explaination of some fields for supporting you to fill out with required and correct data about scanner

SERIAL NUMBER: These are series of characters used to identify a scanner

Tag number:This is a series of characters which identifies each asset of an organization

Entry date in the bank: The date when the bank has received the scanner from supplier

Warranty time(in months): The time given by the supplier to the bank for scanner to be returned back if it doesn't meet the specifications

Life time (in months): The time the scanner is supposed to be used and after that time the computer should not be reliable(provided by the manufacturer and shuould be followed)

Store: If the scanner is in stock which stock where it is stored

Purchase price(RWF): Amount that spent while buying scanner

Sell price: If the scanner was sold, at which price it was sold.

Working status: Behaviour of the scanner while being used

Category type: The type of scanner

Connection type: How the scanner os connected(installed) when it is being used

Host IP: The IP the scanner has when it is connected using network way

Version: The version for the scanner

Other type name: If category type is other, specify that category
