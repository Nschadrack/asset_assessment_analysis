Before filling out the form for recording a new System hosted and application, here is the explaination of some fields for supporting you to fill out with required and correct data about System hosted and application

SERIAL NUMBER: These are series of characters used to identify a System hosted and application

Tag number:This is a series of characters which identifies each asset of an organization

Entry date in the bank: The date when the bank has received the System hosted and application from supplier

Warranty time(in months): The time given by the supplier to the bank for System hosted and application to be returned back if it doesn't meet the specifications

Life time (in months): The time the System hosted and application is supposed to be used and after that time the computer should not be reliable(provided by the manufacturer and shuould be followed)

Store: If the System hosted and application is in stock which stock where it is stored

Purchase price(RWF): Amount that spent while buying System hosted and application

Sell price: If the System hosted and application was sold, at which price it was sold.

Working status: Behaviour of the System hosted and application while being used

Category: The type of System hosted and application whether TV or monitor

System hosted and application size: The size of the System hosted and application in inches
