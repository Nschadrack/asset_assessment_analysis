Before filling out the form for recording a new Note counter, here is the explaination of some fields for supporting you to fill out with required and correct data about Note counter

SERIAL NUMBER: These are series of characters used to identify a Note counter

Tag number:This is a series of characters which identifies each asset of an organization

Entry date in the bank: The date when the bank has received the Note counter from supplier

Warranty time(in months): The time given by the supplier to the bank for Note counter to be returned back if it doesn't meet the specifications

Life time (in months): The time the Note counter is supposed to be used and after that time the computer should not be reliable(provided by the manufacturer and shuould be followed)

Store: If the Note counter is in stock which stock where it is stored

Purchase price(RWF): Amount that spent while buying Note counter

Sell price: If the Note counter was sold, at which price it was sold.

Working status: Behaviour of the Note counter while being used

Category: The type of Note counter whether TV or monitor

Note counter size: The size of the Note counter in inches
