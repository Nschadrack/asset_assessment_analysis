Before filling out the form for recording a new printer, here is the explaination of some fields for supporting you to fill out with required and correct data about printer

SERIAL NUMBER: These are series of characters used to identify a printer

Tag number:This is a series of characters which identifies each asset of an organization

Entry date in the bank: The date when the bank has received the printer from supplier

Warranty time(in months): The time given by the supplier to the bank for printer to be returned back if it doesn't meet the specifications

Life time (in months): The time the printer is supposed to be used and after that time the computer should not be reliable(provided by the manufacturer and shuould be followed)

Store: If the printer is in stock which stock where it is stored

Purchase price(RWF): Amount that spent while buying printer

Sell price: If the printer was sold, at which price it was sold.

Working status: Behaviour of the printer while being used

Category type: The type of printer

Connection type: How the printer os connected(installed) when it is being used

Host IP: The IP the printer has when it is connected using network way

Version: The version for the printer

Other type name: If category type is other, specify that category
